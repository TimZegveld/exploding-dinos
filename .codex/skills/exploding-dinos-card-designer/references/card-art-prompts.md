# Card Art Prompt Templates

Use these templates when generating or asking the user to generate card artwork for Exploding Dinos.

## Full Prompt

```text
Playful prehistoric cartoon card game illustration for a card named "<CARD NAME>".

Scene: <main dinosaur or object action that communicates the card effect>. Use a simple prehistoric setting that supports the gameplay idea without clutter.

Style: chunky shapes, bold clean dark outlines, warm board-game illustration style, expressive dinosaur character, charming and slightly chaotic, high polish but readable at small card size.

Composition: vertical playing-card artwork, central readable subject, strong silhouette, clear foreground action, simple background. Leave clean space near the top and bottom because card title and effect text will be added separately in the game UI.

Color palette: <family colors>, dark brown outlines, cream highlights, warm prehistoric atmosphere.

Constraints: no text, no letters, no numbers, no watermark, no logo, no UI, no card border, no title, no effect text.
```

## Short Prompt

```text
Playful prehistoric cartoon card game illustration, <subject/action>, chunky shapes, bold dark outlines, warm board game art, vertical card composition, clear readable silhouette, simple prehistoric background, no text, no letters, no numbers, no watermark.
```

## Example: Tijdlijn Kneden

```text
Playful prehistoric cartoon card game illustration for a card named "Tijdlijn Kneden".

Scene: a small expressive green dinosaur bends a glowing amber prehistoric timeline like soft clay. The timeline curves through the air as a golden fossil-track ribbon with paw prints, fossils, amber stones, and dust particles swirling around it. The dinosaur looks focused but funny, as if carefully reshaping the future.

Style: chunky shapes, bold clean dark outlines, warm board-game illustration style, expressive dinosaur character, charming and slightly chaotic, high polish but readable at small card size.

Composition: vertical playing-card artwork, central dinosaur character, glowing timeline sweeping diagonally through the image, simple prehistoric valley background with warm rocks and amber light. Leave clean space near the top and bottom because card title and effect text will be added separately in the game UI.

Color palette: amber, warm gold, soft orange, leafy green dinosaur, dark brown outlines, cream highlights.

Constraints: no text, no letters, no numbers, no watermark, no logo, no UI, no card border, no title, no effect text.
```
